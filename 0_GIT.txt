[GIThub]
    1. https://github.com
    2. 버전관리 서비스 (VCS)
    3. 소스코드를 GITHUB 서버에 저장하고 다른 사람과 동시에 병렬 관리가 가능하다.
    4. 특징
        - 소프트웨어 개발에서 소스코드를 효과적으로 관리, 무료
        - 개발 프로젝트내 폴더 작업을 기록하고 버전(과거) 관리, 추적
        - 여러 사람들과 동일한 프로젝트(저장서-리포지토리)를 병렬 작업
    5. 대표 용어
        - Repository : 저장소 / 프로젝트의 파일이나 폴더를 저장하는 저장소
        - Commit : 파일을 추가/ 변경된 내용을 저장소에 저장하는 작업
        - push : 추가/변경된 내용을 원격(github) 저장소에 업로드 작업
    6. 저장소 만드는 사람/계정, 파일 commit/push 이름/이메일(서명) 별로 존재한다
    7.
